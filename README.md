# Geometry Lap Website

Static website for **Geometry Lap** (group of experts in computational geometry and 3D tools), ready to publish with **GitHub Pages**.

## Publish on GitHub Pages

1. Create a new GitHub repository (for example: `geometry-lap`).
2. Upload all files from this folder to the repository root.
3. In GitHub, open **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (or `master`) and folder `/ (root)`
5. Save and wait for deployment.
6. Your site will be available at:
   - `https://<your-github-username>.github.io/<repo-name>/`
